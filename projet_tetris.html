🎮 Projet Tetris – Version Python

📌 Description

Avec un collègue expérimenté, j’ai développé une version basique du célèbre jeu Tetris en Python. Ce projet visait à recréer les principales mécaniques du jeu tout en explorant des concepts fondamentaux de programmation liés au développement de jeux vidéo.

⚙️ Fonctionnalités principales

  • Génération aléatoire des pièces
  • Détection et suppression automatique des lignes complètes
  • Gestion dynamique de la vitesse de descente (score et niveau)
  • Mouvement des pièces : gauche, droite, rotation, descente rapide
  • Interface graphique avec le module FLTK
  • Système de score évolutif avec la difficulté

🚧 Difficultés rencontrées

Le principal défi a été l’optimisation des performances. Le jeu devenait lent à mesure qu’il progressait car la grille se régénérait sans effacer les couches précédentes, causant une surcharge mémoire. Ce problème une fois résolu, le reste du projet s’est déroulé sans encombre.

🎓 Leçons apprises

Ce projet m’a permis de renforcer mes compétences en Python et de mieux comprendre les principes de l’optimisation dans un contexte de jeu vidéo. Il m’a également sensibilisé à l’importance de structurer proprement le code pour maintenir de bonnes performances.

🛠️ Technologies utilisées

Python, FLTK
